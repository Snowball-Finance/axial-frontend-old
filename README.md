## Installation

```bash
$ npm install
```

Create a `.env.local` file with a valid `REACT_APP_NETWORK_URL` (e.g. hardhat, mainnet):

```bash
REACT_APP_NETWORK_URL="http://127.0.0.1:8545/"
```
## Usage

```bash
$ npm run start
```
